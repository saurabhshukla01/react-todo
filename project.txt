Sure! Let's go step by step on **how to create a new React project from scratch**. I’ll explain for beginners, so you can follow easily.

---

## **1. Install Node.js and npm**

React requires **Node.js** and **npm** (Node Package Manager).

1. Check if you have Node.js installed:

```bash
node -v
npm -v
```

2. If not installed, download and install it from [https://nodejs.org](https://nodejs.org).

---

## **2. Create a new React project**

There are a few ways to start a React project. The most common today is using **Vite** or **Create React App (CRA)**.

### **Option A: Using Vite (Recommended for new projects)**

1. Open terminal in your desired folder:

```bash
npm create vite@latest my-app
```

2. You’ll be prompted to choose:

   * **Project name:** `my-app` (or any name you like)
   * **Framework:** `React`
   * **Variant:** `JavaScript` or `TypeScript` (choose TypeScript if you want types)
3. Go into the project folder:

```bash
cd my-app
```

4. Install dependencies:

```bash
npm install
```

5. Run the project:

```bash
npm run dev
```

6. Open browser at the URL it shows (usually `http://localhost:5173/`).

---

### **Option B: Using Create React App (Older but still works)**

1. Open terminal:

```bash
npx create-react-app my-app
```

2. Go into the folder:

```bash
cd my-app
```

3. Start the development server:

```bash
npm start
```

This opens the app in `http://localhost:3000/`.

---

## **3. Project Structure Overview (Vite)**

After setup, your folder looks like this:

```
my-app/
│
├─ index.html          # Main HTML
├─ package.json        # Project config & dependencies
├─ vite.config.js      # Vite config
├─ src/                # All your React code
│   ├─ main.jsx        # Entry point
│   ├─ App.jsx         # Main App component
│   └─ assets/         # Images, CSS
└─ node_modules/       # Installed packages
```

---

## **4. Useful Commands**

| Command                     | Purpose                                     |
| --------------------------- | ------------------------------------------- |
| `npm install`               | Install dependencies                        |
| `npm run dev` / `npm start` | Run development server                      |
| `npm run build`             | Build production-ready code                 |
| `npm run lint`              | Check code formatting (if ESLint installed) |

---

## **5. Optional: Install additional packages**

* **React Router** (for routing):

```bash
npm install react-router-dom
```

* **Axios** (for API calls):

```bash
npm install axios
```

* **Tailwind CSS** (for styling):

```bash
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```
